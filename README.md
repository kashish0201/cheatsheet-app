# 📘 CheatSheet App

An AI-powered cheat sheet generator that lets you upload multiple files (PDF, DOCX, PPTX, code, etc.) and generates a structured cheat sheet from them.

---

## Features

- Upload multiple files at once
- Generate a combined cheat sheet from all uploaded files
- Maintain a session (add more files anytime)
- Prevent duplicate uploads
- Reset session
- Clean and responsive UI
- Fast generation using AI models

---

## Tech Stack

### Frontend
- Next.js
- TypeScript
- CSS

### Backend
- Flask
- Python

### AI
- OpenRouter (Llama 3 / OpenAI-compatible API)
